Aggiornata per supportare la Android 13, quest'applicazione open source vanta le seguenti funzioni:

* Supporto per Dash Clock (si, funziona ancora)
* Notifiche, anche con ripetizione settimanale
* Selezione della lingua preferita
* Widget ridimensionabile e configurabile per la schermata home
* Protezione delle note con password
* Liste personalizzate tra le quali si possono spostare le note
* Liste dove si possono riordinare a piacere le note
* Ricerca tra le note
* Disponibili temi chiari e scuri
* Importazione & esportazione delle note su file JSON
* Sincronizzazione con file ORG nella memoria locale
* Cronologia nota
* Elenco di note eliminate

Se vuoi migliorare quest'app, fare una donazione o condividere qualche idea,
visita la nostra pagina su github: https://github.com/spacecowboy/NotePad
