Open source notepad and to-do list app using the latest Android technologies.

* Dash clock support (yes, it still works)
* Notifications: set reminders at a certain time, with optional repetition certain weekdays
* Language selection
* Widget, with lockscreen support, resizeable and configurable.
* Password-protect important notes
* Move tasks between different lists
* Drag to reorder tasks
* Integration with Android Agenda widget
* Search
* Light and dark themes
* Back up (import and export) notes to a JSON file
* Sync with an ORG file on the local storage

Improve this app, or simply share ideas.
Come visit at https://github.com/spacecowboy/NotePad
